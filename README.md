## 功能介绍（完整见运行截图）
管理员：基本功能包括登录、注销和注册管理系统。可通过各模块对用户、歌手、歌曲、歌单、订单和评论进行管理，如添加、编辑、删除和批量操作。系统首页显示重要统计概况和用户交互数据，利用图表直观呈现用户活跃度及歌曲成功率以优化运营策略。
